# Режим стримера
Иллюзия безопасности... для стримера
![Фильтрация банворда в режиме стримера](/img/streamer-mode.png)
Как вы знаете у нас на сервере разрешены банворды... а что насчет стримеров то, как им быть?

Все просто! Спецаильно для стримеров был создан "режим стримера", фильтрующий все сообщения с банвордами в чате. А для членов администрации он дополнительно цензурит весь админ чат. Все достаточно просто и прозаично.
