# Изменение роста

![Пример разницы роста меж двух игроков с максимальным и минимальным ростом](/img/height.png)

В недавних обновлениях Minecraft Mojang добавили возможность изменять размер модельки игрока. Нам эта идея сразу приглянулась, и мы решили использовать ее как способ самовыражения на сервере

Теперь вы можете указать свой рост из реальной жизни и ваш персонаж в игре станет того же роста. Это простая, но приятная фишка, которая позволяет выделиться и добавить немного уникальности в свой образ

| Команда | Что делает |
| ------- | ---------- |
| `/height set <рост>` | Устанавливает ваш рост в сантиметрах. Например: `/height set 180` |
| `/height reset` | Сбрасывает рост до стандартного значения `183 сантиметра` |
