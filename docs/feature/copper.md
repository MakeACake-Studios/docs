# Окисление меди водой

Ожидание - дело трудное, особенно если это касается окисления меди. Разумеется, можно расставить блоки и уйти по своим делам, но это всё ещё остаётся затратным по времени удовольствием. 
Мы предлагаем вам решение - ручное окисление через бутылочки! Наберите воды в одну и кликните пкм по медному блоку (Не вощёному, очевидно)
