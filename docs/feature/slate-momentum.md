# Моментальное ломание сланца

После получения наилучшего инструментария остаётся лишь один враг комфорта - глубинный камень. Даже с максимально сильной киркой ломание сланца ощущается ужасно долгим. 
Потому мы вмешаемся и привнесём удовольствие в копание!
Теперь, если вы имеете Незеритовую Кирку с Эффективностью V, а так же маяк со вторым уровнем Спешки - Глубинная порода больше не будет для вас проблемой
