# Статрэк

Ещё один способ покрасоваться циферками, но теперь уже на собственных инструментах. За небольшое количество вечно-золотых можно приобрести "Статрэк" на выбранный вами инструмент, в добавок получая цветное переименование, чтобы закрепить уникальность избранного вами предмета
