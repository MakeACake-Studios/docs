# Бесконечные торги с жителями

![Пример неограниченных торгов с жителями](/img/infinite-trades.png)

Иногда торговля с жителями превращается в ужас: сильный подъём цен, дневные лимиты на торговлю и прочее...
Мы упростили это, позволив игрокам торговать с жителями без лимита по предметам, а также без повышения цен из-за количества купленных предметов.

Теперь задачи по сбору шалкера пузырьков опыта или прокачке всех своих жителей до максимального уровня не являются непростым испытанием для терпения.
