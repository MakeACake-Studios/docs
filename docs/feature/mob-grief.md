# Гриф мобов
На сервере отключен гриф со стороны криперов, гастов, эндерменов и... эндер дракона.

![Крипер пытающийся подорвать спавн (кто это вообще читает?)](/img/mob-grief.png)

Что это значит? Их взрывы по-прежнему **наносят урон игрокам и мобам**, но **не разрушают блоки в радиусе**. То есть если крипер взорвался у тебя на ферме - ничего страшного не произойдет, кроме, может быть, нервного тика

Это решение сделано для того, чтобы сохранить важные постройки игроков и не вынуждать всех жить в обсидиановых бункерах.
