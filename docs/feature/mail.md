# Мейлы

Ваш друг ушёл, не сказав ни слова, а у вас для него важное сообщение? На сервере появилась возможность отправлять письма тем, кого на данный момент в сети нет, которое человек получит при следующем входе в игру. Отправлять/получать/пересматривать полученные сообщения можно через команду /mail
